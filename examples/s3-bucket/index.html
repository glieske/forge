forge repository

Static plugin and update repository layout for the forge CLI.

Configure forge

forge config set repositories.plugins_url https://d3r3o2lmc0s5i7.cloudfront.net/forge/plugins
forge config set repositories.updates_url https://d3r3o2lmc0s5i7.cloudfront.net/forge/updates
forge self-update channel stable

Repository indexes

  • Plugin signing public key
  • Plugin index
  • connect plugin versions
  • connect 1.2.0 manifest
  • Stable update channel
  • Beta update channel
  • Dev update channel

        forge_<os>_<arch>.tar.gz
        forge_<os>_<arch>.zip
